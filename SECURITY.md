# Security Policy

## Supported Versions

The following versions of sleek are provided with security updates.

| Version | Supported |
|---------|-----------|
| 2.x.x   | ✅         |
| 1.x.x   | ❌         |
| 0.x.x   | ❌         |

## Reporting a Vulnerability

If you find a vulnerability in sleek, please send as much detail as possible to ransome@mailbox.org.
